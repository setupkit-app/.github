  ## The problem

  Most people use Claude Code or Cursor without ever writing a config file, then complain the AI keeps suggesting React patterns in a Vue project, or pushes a testing library they don't use.    
   
  A config file fixes that. But writing a good one is tedious, and copying random templates from GitHub rarely fits your context.                                                                 
                                                            
  setupkit.app asks you a few questions and outputs a file you can drop into your repo.                                                                                                           
                                                            
  ## What's inside                                                                                                                                                                                
                                                            
  - A questionnaire covering stack, conventions, level, project type                                                                                                                              
  - One input, files for all four major AI coding tools     
  - Examples per stack: Next.js, Django, Flutter, Go, Laravel, FastAPI, Astro                                                                                                                     
  - 3 free files, paid plan for unlimited
